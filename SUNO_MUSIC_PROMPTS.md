## Usage Instructions:

1. Go to [suno.ai](https://suno.ai)
2. Copy the prompt text for each track
3. Set the style/mood tags as suggested
4. Generate and download the audio
5. Convert to MP3 or OGG format
6. We can integrate them into the game using the Web Audio API

---

## Integration Notes:

Once you have the Suno-generated tracks, I can:
- Encode them as base64 and embed directly in the HTML
- Or set up audio file loading from a URL/local path
